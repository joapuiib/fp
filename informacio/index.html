    Informació

    Llicència

    Aquest material està llicenciat baix la llicència Creative Commons Reconeixement-NoComercial-CompartirIgual 4.0 Internacional.

    Correccions

    Si trobeu alguna errada en el material, podeu comunicar-la mitjançant l'apartat Issues al repositori de GitHub indicant:

    1. L'enllaç a la pàgina on s'ha trobat l'errada.
    2. La correcció a realitzar.

    Suggeriments

    Si voleu fer alguna suggerència o millora, podeu fer-ho mitjançant l'apartat Discussions al repositori de GitHub.

    Contribucions

    Aquest repositori accepta contribucions.

    La majoria dels apunts estan allotjats en repositoris privats, per tant, no es pot contribuir directament a aquests. En aquests casos, es pot obrir una Incidència o Discussió per aportar suggeriments o correccions.

    En els repositoris públics, es pot contribuir al material directament mitjançant una Pull Request.

    Desenvolupament

    Aquest lloc web està desenvolupat amb les següents eines i llibreries:

    • MkDocs: Generador de llocs web estàtics amb Markdown i Python.
    • Material for MkDocs: Tema per a MkDocs.
    • Python Markdown: Processador de Markdown per a Python.
    • PyMdown Extensions: Extensions per a Python Markdown.
    • Pygments: Ressaltat de sintaxi de codi amb Python.
    • Git: Sistema de control de versions.
    • GitHub Pages: Servei de pàgines web estàtiques de GitHub.
    • GitHub Actions: Automatització de tasques de GitHub.
